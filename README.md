# Ayoba Hackerthon

## Getting started

```cli
git clone https://github.com/mindsgn/ayoba-gamejam
```

```cli
cd ayoba-gamejam

```

```cli
yarn

```

```cli
yarn install

```

```cli
yarn start:dev

```
